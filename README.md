# go-to-the-future
A command-line tool to establish a temporal tunnel and transport you to the future.

## Usage

> `gtf` is abbr of "go to the future".

```sh
gtf <duration>
```

for example:

```sh
gtf 5s
```

then you will be transported to the future for 5 seconds after the command is finished.

If your target time is very long, the progress will take a long time to finish.
